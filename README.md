# Adaptation

> every hit strengthens the machine.

Adaptation is an ULTRAKILL mod inspired by Mahoraga from Jujutsu Kaisen.

Taking damage causes V1 to adapt:
- incoming damage is reduced
- outgoing damage is increased
- adaptation stacks infinitely

---

## Features

- adaptation system
- Configurable adaptation percentages
- Damage resistance scaling
- Damage boost scaling
- Mahoraga-inspired adaptation wheel
- Speed adaptation was discontinued after V1 became faster than the game could process speed and movement
